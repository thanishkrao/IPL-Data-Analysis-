# 🏏 IPL Data Analysis Project

This project performs Exploratory Data Analysis (EDA) on a ball-by-ball IPL dataset using Python.

The objective of this project is to analyze player and team performance trends from historical IPL data and generate meaningful insights using data visualization.

---

## 📊 Project Objectives

- Identify the Top 10 Run Scorers in IPL history
- Find the Top 10 Six Hitters
- Determine the Top 10 Wicket Takers
- Analyze the Most Successful IPL Teams based on match wins

---

## 🛠️ Technologies Used

- Python
- Pandas
- Matplotlib

---

## 📂 Dataset Information

- The dataset used in this project is a ball-by-ball IPL dataset.
- Source: Kaggle IPL Dataset
- The dataset contains over 278,000 rows and 60+ columns including:
  - Batting team
  - Bowling team
  - Batter
  - Bowler
  - Runs scored
  - Wickets
  - Match details
  - Season information

(Note: The dataset file is not uploaded due to GitHub size limitations.)

## 📈 Key Insights

### 🏆 Top Run Scorers
Virat Kohli, Rohit Sharma, Shikhar Dhawan and other consistent performers dominate the run charts.

### 💥 Most Six Hitters
Power hitters like Chris Gayle and AB de Villiers lead in sixes.

### 🎯 Top Wicket Takers
Leading bowlers consistently take wickets across seasons.

### 🥇 Most Successful Teams
Teams like Mumbai Indians and Chennai Super Kings are among the most successful franchises in IPL history.

## 🚀 How to Run the Project

1. Clone the repository:
   ```
   git clone https://github.com/thanishkrao/IPL-Data-Analysis-.git
   ```

2. Install required libraries:
   ```
   pip install pandas matplotlib
   ```

3. Place the IPL dataset file (`IPL.csv`) inside the project folder.

4. Run the script:
   ```
   python ipl_analysis.py
   ```
